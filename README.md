
# mdx-docs

Document and develop React components with MDX

```sh
npm i -D mdx-docs
```

- Create documentation with markdown
- Import and use React components
- Zero-config
- No plugins
- Component-based API
- Customizable themes

## Getting Started

- install
- package run scripts
- mkdir docs
- vi docs/index.mdx
- npm start
